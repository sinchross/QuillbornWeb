# Quillborn Site

Workspace do site oficial do Quillborn.

Arquivos iniciais:
- `docs/site-plan.md`: planejamento completo do site
- `docs/asset-production-plan.md`: plano de capturas, imagens e vídeos vindos do app
- `assets/screenshots/.gitkeep`: pasta reservada para screenshots do Quillborn
- `assets/videos/.gitkeep`: pasta reservada para vídeos curtos e demo reels
- `assets/brand/.gitkeep`: pasta reservada para logo, ícones e peças visuais
